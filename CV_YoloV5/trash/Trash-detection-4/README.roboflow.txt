
Trash-detection - v4 trash-detection-v4
==============================

This dataset was exported via roboflow.ai on March 8, 2022 at 8:45 PM GMT

It includes 1499 images.
Trash are annotated in YOLO v5 PyTorch format.

The following pre-processing was applied to each image:
* Resize to 750x750 (Stretch)

The following augmentation was applied to create 1 versions of each source image:
* 50% probability of horizontal flip
